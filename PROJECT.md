## 一、项目概述

**BitHabit** 是一个帮助学生培养学习习惯、管理暑假作业的 PWA 应用。

- **产品形态**：PWA（Progressive Web App）
- **核心价值**：一键链接分享 → 浏览器打开 → 添加到主屏幕 → 即获得 App 体验，无需安装
- **离线能力**：Service Worker + IndexedDB，断网可打卡，联网自动同步
- **目标用户**：中学生，可分享链接给同班同学共同使用

---

## 二、核心功能（MVP 优先级）

### P0：自动作业计划生成（MVP 第一版）
- 用户录入各科作业（精确到卷/页/套）
- 设定暑假时间区间、每日学习时长上限、休息日、假节日
- 系统智能分配每日任务，生成学习日历
- 用户可手动调整生成后的计划

### P1：进度追踪与追赶
- 每日打卡确认完成
- 未完成任务自动重新分摊到后续日期

### P2：社交监督
- 同学之间可见完成进度（不暴露具体内容）
- 互相激励

## 三、MVP 详细设计：自动作业计划生成

### 3.1 作业录入

提供两种录入方式，用户可任选或混用：

#### A1：手动录入

逐条添加作业，支持快速连录（「继续添加」按钮）。

| 字段 | 说明 | 必填 |
|------|------|------|
| 科目 | 下拉预设 + 可自定义 | ✅ |
| 任务类型 | 练习册/模拟卷/单词/阅读理解/自定义 | ✅ |
| 总量 | 数字 + 单位（页/套/篇/个） | ✅ |
| 单位耗时 | 每单位预估分钟数 | 选填（有默认值） |
| 备注 | 自由文本 | 选填 |

#### A2：AI 自然语言录入

用户用自然语言描述作业 → DeepSeek API 解析 → 返回结构化任务列表 → 用户确认/编辑后保存。

**流程**：
```
输入自然语言描述 → 点击「AI 解析」
    → 后端 PHP 代理调用 DeepSeek API（Key 存服务端）
    → 返回结构化任务预览列表
    → 用户确认（可编辑/删除/手动补充）→ 保存
    → 解析失败则保留原始文本，降级为手动录入
```

**DeepSeek API**：
- 前端调 `/api/ai/parse`，PHP 后端代理转发
- API Key 仅在服务端，前端不可见
- 需加频率限制防滥用

### 3.2 日程配置（独立页面）

日程配置独立于作业计划，可复用（暑假、学期中均可使用）。

#### 每周固定

| 星期 | 时间段 | 标签 |
|------|--------|------|
| 周一 | 8:00-14:00 | 补习班 |
| 周三 | 8:00-14:00 | 补习班 |
| 周五 | 8:00-14:00 | 补习班 |
| 周日 | 全天 | 休息日 |

- 支持添加多条、删除、编辑
- 时间段可选「全天」或具体时段
- 标签为自由文本（补习班/休息日/游泳课…）

#### 特殊日期

| 日期范围 | 时间段 | 标签 |
|----------|--------|------|
| 7/15 - 7/18 | 全天 | 旅行 |
| 7/20 | 14:00-18:00 | 有事 |

- 单日或日期范围
- 同样可选全天/具体时段

#### AI 辅助录入

同样支持自然语言解析（`/api/ai/parse-schedule`），流程同作业录入：输入描述 → AI 解析 → 确认后保存。

### 3.3 计划约束设定

生成计划时，系统自动引用日程配置。用户只需设定：

| 字段 | 说明 |
|------|------|
| 暑假起止日期 | 开始日 → 结束日 |
| 每日可用时段 | 默认 8:00-22:00（可调整） |
| 分配策略 | 平均 / 前紧后松 / 智能权重（默认） |

页面底部展示日程配置摘要，点击可跳转修改。

### 3.4 平均分配策略（v1）

**逻辑**：总工作量 ÷ 可用天数 = 每天等量分配。

1. 计算总工作分钟数（Σ 总量 × 单位耗时）
2. 计算可用天数（暑假区间 - 不可用时段占用的天数 - 休息日）
3. 每天配额 = 总分钟数 ÷ 可用天数
4. 按每天配额，从各科均匀取量填入每日计划
5. 如总工时超出可用时间，提示用户减少作业或延长时间

后续迭代：前紧后松（v2）→ 智能权重+科目轮换（v3）

### 3.5 任务时间编排

- 系统为每个任务指定**具体时间段**（如 8:00-8:20 数学）
- **核心交互：拖拽调整**
  - 在日视图时间轴上，用户可直接拖拽任务卡片移动时间段
  - 拖拽改变任务时长（拉伸卡片边缘调整时长）
  - 支持跨时段拖拽重排序
- 也支持点击进入编辑面板，精确设定时间

### 3.6 计划输出

- **日历视图**：月视图 + 日视图
- **每日清单**：列出当天所有任务及预估耗时
- **导出/分享**：可将计划分享给同学

## 五、部署架构

```
同学手机浏览器
       │
  https://bithabit.dpdns.org
       │
  Cloudflare Tunnel (Docker @ QNAP)
       │
  QNAP 虚拟主机 :1110
       │
  /BitHabit/Web/ (PHP + 前端资源)
       │
  MariaDB 10 (/var/run/mariadb10.sock)
```

- 前端 PWA 和后端 API 统一通过域名访问
- Cloudflare 自动提供 HTTPS 证书
- 同学无需任何特殊配置，浏览器打开链接即可
